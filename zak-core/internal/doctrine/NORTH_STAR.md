# ZAK Doctrine: The North Star

## The Axiom

> "ZAK defines the only place in a system where authority is allowed to exist without becoming liability."

## Implications

1.  **Authority is Physics, Not Permission**: We do not ask the model for permission to be safe. We enforce safety as a physical constraint (The Cage).
2.  **Liability is Unbounded Context**: If context exists outside the Cage, it is a liability. ZAK eliminates liability by evaporating context.
3.  **The Kernel is the Boundary**: Anything inside the kernel is deterministic authority. Anything outside is probabilistic noise.
